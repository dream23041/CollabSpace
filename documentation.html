🚀 Быстрый старт

Добро пожаловать в CollabSpace! Это руководство поможет вам быстро начать работу с платформой.

💡 Совет: Для начала работы вам понадобится только веб-браузер и учетная запись CollabSpace.

Шаги для начала работы:

  1. Зарегистрируйте аккаунт на главной странице
  2. Войдите в систему используя ваши учетные данные
  3. Создайте рабочее пространство для вашего проекта
  4. Пригласите collaborators для совместной работы
  5. Начните создавать файлы и работать вместе!

📝 Регистрация аккаунта

Создание учетной записи в CollabSpace занимает всего несколько минут.

Требования к учетной записи:

  • Имя пользователя: 3-20 символов (буквы, цифры, подчеркивания)
  • Email: действительный email адрес для восстановления
  • Пароль: минимум 6 символов
Пример правильного имени пользователя:
- my_username
- developer123
- john_doe

Неправильные имена:
- ab (слишком короткое)
- username! (недопустимые символы)
- very_long_username_here (слишком длинное)

🛠 Создание первого проекта

После входа в систему вы попадете в личный кабинет. Вот как создать ваш первый проект:

Создание рабочего пространства:

  1. Нажмите кнопку "Новое пространство" в боковой панели
  2. Введите название рабочего пространства
  3. Добавьте описание (опционально)
  4. Нажмите "Создать"

Создание первого файла:

  1. В рабочем пространстве нажмите "Создать файл"
  2. Выберите тип файла (JavaScript, HTML, CSS, текст и т.д.)
  3. Введите название файла
  4. Нажмите "Создать"

✏ Основы редактора

Редактор CollabSpace предоставляет все необходимые инструменты для эффективной работы с кодом и текстом.

Интерфейс редактора:

  • Панель инструментов - быстрые действия и форматирование
  • Область редактирования - основное рабочее пространство
  • Дерево файлов - навигация по файлам проекта
  • Информационная панель - сведения о текущем файле
⚠ Важно: Все изменения автоматически сохраняются каждые 30 секунд. Вы также можете сохранить вручную с помощью Ctrl+S.

Основные функции редактора:

📝 Текстовое редактирование

Полнофункциональный текстовый редактор с поддержкой Markdown

🎨 Подсветка синтаксиса

Автоматическая подсветка для всех популярных языков программирования

🔍 Поиск и замена

Быстрый поиск по файлу с функцией замены

💾 Автосохранение

Автоматическое сохранение каждые 30 секунд

📊 Поддерживаемые типы файлов

CollabSpace поддерживает различные типы файлов с соответствующей подсветкой синтаксиса.

Тип файла Расширение Особенности
JavaScript .js Подсветка ES6+, автодополнение
HTML .html Подсветка тегов, валидация
CSS .css Подсветка свойств, автодополнение
Текстовый файл .txt Простое текстовое редактирование
Markdown .md Поддержка разметки, предпросмотр
JSON .json Валидация JSON, форматирование
💡 Подсказка: Вы можете изменить тип файла после создания через меню "Переименовать".

👥 Приглашение пользователей

CollabSpace позволяет приглашать других пользователей для совместной работы над проектами.

Способы приглашения:

📧 По ссылке

Создайте уникальную ссылку-приглашение

📧 По email

Отправьте приглашение на email

🔗 Публичная ссылка

Создайте публичный доступ к проекту

Как пригласить пользователя:

  1. Откройте нужное рабочее пространство
  2. Нажмите кнопку "Пригласить" в верхней панели

⚡ Редактирование в реальном времени

CollabSpace позволяет нескольким пользователям одновременно редактировать файлы с синхронизацией в реальном времени.

Особенности совместного редактирования:

  • Мгновенная синхронизация - изменения видны сразу всем участникам
  • Цветные курсоры - каждый пользователь имеет свой цвет курсора
  • Конфликт-фри редактирование - система автоматически разрешает конфликты
  • История изменений - полная история правок каждого пользователя

Индикаторы совместной работы:

  • 🌈 Цветные аватары - показывают кто сейчас онлайн
  • 👆 Цветные курсоры - показывают где редактируют другие
  • 📝 Выделения - показывают активные правки других пользователей
  • 💬 Всплывающие подсказки - показывают кто делает изменения

Функции чата:

Как использовать чат:

🏢 Рабочие пространства

Рабочие пространства позволяют организовать ваши проекты и контролировать доступ к ним.

Создание рабочего пространства:

  1. В личном кабинете нажмите "Новое пространство"
  2. Введите название и описание
  3. Выберите уровень доступа (приватный/публичный)
  4. Нажмите "Создать"

Управление доступом:

Уровень доступа Описание Возможности
Владелец Создатель пространства Все действия, включая удаление
Редактор Приглашенный пользователь Создание и редактирование файлов
Читатель Пользователь с доступом только для чтения Просмотр файлов без возможности редактирования

📤 Импорт и экспорт

Легко переносите ваши проекты между CollabSpace и локальной файловой системой.

Импорт файлов:

  1. В рабочем пространстве нажмите "Импорт файлов"
  2. Выберите файлы на вашем компьютере
  3. Дождитесь завершения загрузки
  4. Файлы появятся в вашем проекте

Экспорт файлов:

  1. Выберите файлы для экспорта
  2. Нажмите "Экспорт" в меню действий
  3. Выберите формат экспорта (ZIP или отдельные файлы)
  4. Скачайте архив на ваш компьютер
💡 Подсказка: Вы можете экспортировать целое рабочее пространство как ZIP-архив, содержащий все файлы и структуру папок.

Поддерживаемые форматы импорта:

  • .txt - текстовые файлы
  • .js - JavaScript файлы
  • .html - HTML документы
  • .css - CSS стили
  • .md - Markdown файлы
  • .json - JSON данные

🔐 Безопасность

CollabSpace обеспечивает защиту ваших данных и проектов.

Меры безопасности:

  • Шифрование данных - все данные передаются по защищенному соединению
  • Аутентификация - безопасный вход с проверкой учетных данных
  • Контроль доступа - гибкая система прав для каждого пользователя
  • Резервное копирование - регулярное сохранение данных
  • Мониторинг - отслеживание подозрительной активности

Рекомендации по безопасности:

  1. Используйте сложные пароли
  2. Не передавайте свои учетные данные
  3. Регулярно обновляйте пароль
  4. Контролируйте доступ к вашим проектам
  5. Сообщайте о подозрительной активности

🔧 Решение проблем

В этом разделе собраны решения распространенных проблем.

Частые проблемы и решения:

❌ Не могу войти в аккаунт

  • Проверьте правильность ввода логина и пароля
  • Убедитесь, что Caps Lock выключен
  • Попробуйте восстановить пароль

❌ Файлы не сохраняются

  • Проверьте подключение к интернету
  • Обновите страницу (F5)
  • Попробуйте сохранить файл вручную (Ctrl+S)

❌ Не вижу изменения других пользователей

  • Обновите страницу
  • Проверьте, что вы в том же рабочем пространстве
  • Убедитесь, что пользователь сохранил изменения

❌ Не могу пригласить пользователя

  • Проверьте, что пользователь зарегистрирован в системе
  • Убедитесь, что у вас есть права на приглашение
  • Попробуйте отправить приглашение по email
📞 Нужна помощь? Если вы не нашли решение вашей проблемы, обратитесь в службу поддержки.
